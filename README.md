# 漫才生成器 (Manzai Generator)

这是一个用于生成漫才风格对话的浏览器用户脚本。

## 功能特点

- 自动生成漫才风格的搞笑对话
- 支持浏览器扩展（如 Tampermonkey）
- 简单易用的界面

## 安装说明

1. 首先安装 Tampermonkey 或类似的用户脚本管理器
2. 点击以下链接安装脚本：
   - [漫才生成器](index.user.js)
   - [漫才对话生成](manzai.user.js)

## 使用方法

1. 安装完成后访问支持的网站
2. 点击界面上的生成按钮
3. 享受生成的漫才对话
